# Template de Iteração

Copiar este bloco para `tasks/todo.md` em tarefas não triviais.

```md
### YYYY-MM-DDTHH:mm:ss+TZ — <título>

**Pedido:**
- <pedido original resumido>

**Objetivo:**
- <resultado esperado>

**Risco:**
- Baixo / Médio / Alto / Crítico

**Papéis IA ativados:**
- Product Owner técnico: <sim/não>
- Arquiteto: <sim/não>
- Frontend: <sim/não>
- Backend: <sim/não>
- Base de dados: <sim/não>
- QA: <sim/não>
- Segurança: <sim/não>
- DevOps: <sim/não>
- Documentação: <sim/não>
- Reviewer: <sim/não>

**Plano:**
- [ ] <passo>

**Ficheiros previstos:**
- `<path>` — <motivo>

**Alterações feitas:**
- `<path>` — <descrição>

**Validação:**
- `<comando>` — <resultado>
- Não executado: <motivo>

**Auditoria de limpeza:**
- Removidos: `<path>` — <motivo ou N/A>
- Mantidos: `<path>` — <motivo ou N/A>
- Candidatos a confirmação: `<path>` — <risco ou N/A>

**Decisões:**
- <decisão ou N/A>

**Lições:**
- <aprendizagem ou N/A>

**Estado final:**
- Concluído / Parcial / Bloqueado
```
