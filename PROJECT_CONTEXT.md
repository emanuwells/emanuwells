# Contexto do projeto

## Identidade

`emanuwells` é simultaneamente o repositório do GitHub Profile README de Emanuel Wells e a fonte do portefólio publicado em `emanuwells.vercel.app`.

## Responsabilidades

- apresentar experiência, projetos e competências;
- dar destaque editorial a casos de trabalho e projetos relevantes;
- manter o caso Maia como narrativa técnica pessoal;
- encaminhar para repositórios e produtos independentes, incluindo o WELLS_OS.

## Fora de âmbito

- autenticação administrativa;
- consola operacional;
- segredos ou configuração de servidores;
- acesso SSH;
- execução de jobs ou alterações de produção.

## Stack

Next.js, React, TypeScript e CSS/Tailwind, com publicação prevista no Vercel.
