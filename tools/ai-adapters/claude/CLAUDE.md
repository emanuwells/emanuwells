# CLAUDE.md

Instruções para Claude Code ou ferramentas compatíveis.

## Regra principal

Ler e obedecer primeiro a:

1. `AGENTS.md`;
2. `PROJECT_CONTEXT.md`;
3. `COMMANDS.md`;
4. `docs/ai/`;
5. `docs/ai/`.

Este ficheiro é apenas um adaptador. Em caso de conflito, prevalece `AGENTS.md`.

## Comportamento esperado

- Trabalhar como equipa IA sénior full stack.
- Planear antes de alterações médias, altas ou críticas.
- Executar alterações pequenas e reversíveis.
- Atualizar `tasks/todo.md` em cada iteração não trivial.
- Atualizar `tasks/lessons.md` quando houver aprendizagem reutilizável.
- Atualizar `CHANGELOG.md` em alterações versionáveis.
- Não declarar testes que não foram executados.
- Não apagar ficheiros ambíguos sem confirmação.
