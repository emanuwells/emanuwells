# Adaptador Devin Desktop

Este diretório existe apenas para orientar ferramentas agentic tipo Devin Desktop.

## Regra

O agente deve tratar `AGENTS.md`, `docs/ai/` e `docs/ai/` como fonte de verdade. Este adaptador não deve conter regras contraditórias.

## Fluxo recomendado

1. Criar plano.
2. Trabalhar numa branch ou worktree.
3. Executar uma fase de cada vez.
4. Correr validações.
5. Produzir relatório final com ficheiros alterados, validações, riscos e rollback.
