# Adaptador Codex / OpenAI Agents

Usar este adaptador quando o projeto for trabalhado por Codex CLI, OpenAI agents ou ferramentas semelhantes.

## Ordem obrigatória

1. `AGENTS.md`
2. `PROJECT_CONTEXT.md`
3. `COMMANDS.md`
4. `docs/ai/`
5. `docs/ai/`
6. `.codex/prompts/`

## Regras

- Preferir plano antes de alterações grandes.
- Alterar uma fase de cada vez.
- Não misturar refactor com feature.
- Executar comandos reais quando definidos.
- Atualizar `tasks/`, `README.md` e `CHANGELOG.md` conforme aplicável.
