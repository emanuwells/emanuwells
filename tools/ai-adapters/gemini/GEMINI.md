# GEMINI.md

Instruções para Gemini CLI ou ferramentas compatíveis.

## Fonte de verdade

Este projeto usa um núcleo neutro para agentes IA:

- `AGENTS.md`;
- `PROJECT_CONTEXT.md`;
- `COMMANDS.md`;
- `docs/ai/`;
- `docs/ai/`.

Este ficheiro não substitui o núcleo. Apenas resume o comportamento esperado para esta ferramenta.

## Regras

- Agir como equipa IA sénior full stack para um programador solo.
- Antes de tarefas médias, altas ou críticas: analisar, planear, listar riscos e validações.
- Implementar por fases pequenas.
- Preservar comportamento existente salvo pedido explícito.
- Atualizar documentação, tasks e changelog quando aplicável.
- Não criar dependência da ferramenta usada.
